Experiment 2 - Living Impressions

Description

In this project I'll use p5.js to create a generated image for this photo.

This is the photo I took in Lake Tahoe.

Technical

Relation to inspring image:

  • Color: The sky, lake, and background mountain colors were sampled from the image using the Photoshop eyedropper tool.
  • Primitives: The stars and waves are rendered with ellipse primitives.
  • Polygons: The mountain is a polygon using Perlin noise generation.
  • Composition: Different random seed values lead to different shapes of mountains and different stars.
  • Life: The waves are animated and the direction of the waves can be manipulated with the mouse cursor.

Reflection

Experiment 2 went pretty smoothly, mainly because there are very good tutorials online for the effects I wanted to do, such as the gradient color background. The one I felt I had a hard time with was making random mountains using perlin noise, which took me quite a bit of time.
